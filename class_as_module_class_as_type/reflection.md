# Рефлексия.
Не понял почему в примерах специализация, например, электровелосипед наследник велосипеда дополненный мотором, представлена в качестве расширения. Класс кончено расшиерился в плане полей, но расширение не как в задаинии: "Расширение класса-родителя (наследник задаёт более общий случай родителя)". Тут явное: "Специализация класса-родителя (наследник задаёт более специализированный случай родителя)".
