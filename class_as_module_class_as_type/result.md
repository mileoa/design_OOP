# Класс как модуль
В python концепция "Класс как модуль" реализована следующим образом. Все файлы .py являются модулями. Модули могут объеденяться в пакеты - папки содержащие в корне __init__.py. Судя по всему иногда можно рассматривать сам файл .py как класс экземпляр которого мы не можем создать - статический класс. Сам файл .py может сожержать сколько угодно определений классов как и сами классы могут содержать сколько угодно определений других классов внутри себя.
