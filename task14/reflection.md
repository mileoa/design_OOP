# Рефлексия
В примере с python дана идея с вызовом исключений при попытке вызвать "переопрделенные" методы. Т.е. был открытый метода a(), мы создаем закрытый метод  _a() или __a() и наследуем его поведение из родителя через вызов super внутри тела метода. Затем переопределяем открытый метод a() и вызываем в нем исключение при вызове.
